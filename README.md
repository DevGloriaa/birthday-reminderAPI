The app allows users to **register, login, and keep track of birthdays** of their friends and family.

## Features ‚ú®

- **User Authentication**
  - Registration with `firstName`, `lastName`, `email`, and `password`

  - Login and session management
- **Birthday Management**
  - Add, view, and update birthdays (planned for future versions)
- **MongoDB Integration**
  - Stores user and birthday data
- **RESTful API**
  - Ready for frontend integration (React)
